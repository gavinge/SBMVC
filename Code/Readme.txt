SBMVC V1.1

作者：Gavin geshan@hotmail.com  from CHINA


为什么要开源这个？

做为一个C++工程师，无论你傲气也好，牛逼也罢。但你不能不遵循好的设计、架构原理,这无关个性,关于对自己的严格和对项目负责.
笔者在中国某知名软件公司工作,该厂也号称客户端方面在业界数一数二,笔者也开发和维护过几十个项目.但也同样发现一个严重的问题:客
户端研发由于需要快速迭代的满足某个方面需求,而往往忽略了代码的良好设计和追求较好的代码架构.一个产品换了不
同研发人员往往代码风格和质量参差不齐.尤其面对代码量巨多的软件,仅仅满足了需求远远不足以解决长远问题.而这恰恰是项目技术leader
需要深度考虑的问题,要么边做变重构(你有那么自觉么?),要么一开始就把代码逻辑化,让代码有标准.

这套开源的简单的基于MVC的框架(适用于windows下VC++,其它系统以后会逐渐更新).就是在笔者维护某大型客户端软件(代码量超过70万行)
在重构该产品是应运而生的,且成功的在新版本重构中使用. 

开源这套东西,也是希望与同行多交流,多学习！



版权:任何商业,非商业都可使用,当然如果能尊重作者的辛劳,部分注释不动最好.
